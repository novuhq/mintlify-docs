Novu's `@novu/js` JavaScript SDK lets you embed a fully-featured notification inbox inside any Angular application. Because Novu doesn't currently ship a native Angular component, this guide uses `NovuUI` from `@novu/js/ui` and Angular's `AfterViewInit` lifecycle hook to mount the Inbox onto a DOM element.

## Prerequisites

- A [Novu account](https://dashboard.novu.co/auth/sign-up) — create one for free or [sign in](https://dashboard.novu.co/auth/sign-in) if you already have one.
- Node.js 18+ and the [Angular CLI](https://angular.dev/tools/cli/setup-local#install-the-angular-cli) installed.

## Setup

<Steps>

### Create an Angular application

Scaffold a new Angular project:

```bash
ng new novu-inbox-angular
cd novu-inbox-angular
```

<Info>
  Novu uses `@novu/js` for Angular. There is no native Angular-specific Novu package — the JavaScript SDK handles mounting via the DOM.
</Info>

### Install `@novu/js`

Add the Novu JavaScript SDK to your project:

<CodeGroup>
```bash npm
npm install @novu/js
```
```bash pnpm
pnpm add @novu/js
```
```bash yarn
yarn add @novu/js
```
```bash bun
bun add @novu/js
```
</CodeGroup>

### Update `app.component.ts`

Open `src/app/app.ts` (or `app.component.ts` in older Angular versions) and implement `AfterViewInit` to mount the Novu Inbox once the view is ready:

```typescript app.component.ts
import { Component, ViewChild, ElementRef, AfterViewInit } from '@angular/core';
import { RouterOutlet } from '@angular/router';
import { NovuUI } from '@novu/js/ui';

@Component({
  selector: 'app-root',
  standalone: true,
  templateUrl: './app.html',
  styleUrl: './app.css'
})
export class App implements AfterViewInit {
  @ViewChild('novuInbox') novuInbox!: ElementRef<HTMLElement>;
  title = 'novu-angular';

  ngAfterViewInit() {
    const novu = new NovuUI({
      options: {
        applicationIdentifier: 'YOUR_APPLICATION_IDENTIFIER',
        subscriberId: 'YOUR_SUBSCRIBER_ID',
      },
    });

    novu.mountComponent({
      name: 'Inbox',
      props: {},
      element: this.novuInbox.nativeElement,
    });
  }
}
```

<Note>
  Replace `YOUR_APPLICATION_IDENTIFIER` with the value from **API Keys** in your Novu dashboard, and `YOUR_SUBSCRIBER_ID` with your user's unique ID. Novu automatically creates a subscriber if the provided ID doesn't exist yet.
</Note>

**Finding your credentials:**

- **`applicationIdentifier`** — In the [Novu dashboard](https://dashboard.novu.co), click **API Keys** and copy your Application Identifier.
- **`subscriberId`** — This represents a user in your system, typically their database user ID. For testing, use the auto-generated ID from the **Subscribers** tab in the dashboard.

### Add the mount point to your template

Open `src/app/app.html` and add a `#novuInbox` template reference variable at the top of the file. The `@ViewChild` decorator in your component class uses this reference to get a handle on the DOM element:

```html src/app/app.html
<div #novuInbox></div>
```

### Run your application

Start the development server:

<CodeGroup>
```bash npm
npm run start
```
```bash pnpm
pnpm start
```
```bash yarn
yarn start
```
```bash bun
bun start
```
</CodeGroup>

Once running, a **bell icon** appears in the top-left corner of the screen. Click it to open the notification inbox. There are no notifications yet — let's trigger one.

### Trigger your first notification

Create a workflow in the Novu dashboard and send a test notification:

1. Go to your [Novu dashboard](https://dashboard.novu.co).
2. Click **Workflows** in the sidebar.
3. Click **Create Workflow** and enter a name (e.g., `Welcome Notification`).
4. Click **Create Workflow** to save it.
5. Click the **Add Step** icon in the workflow editor and select **In-App** as the step type.
6. In the In-App template editor, set:
   - **Subject**: `Welcome to Novu`
   - **Body**: `Hello, world!`
7. Close the editor, then click **Trigger**.
8. Click **Test Workflow**.

### View the notification in your app

Switch back to your Angular app and click the bell icon.

You should see the notification you just triggered. 🎉

</Steps>

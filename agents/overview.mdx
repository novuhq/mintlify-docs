Novu Connect lets you bring AI agents into the messaging platforms where your users already work. Built on the Agent Communication Infrastructure (ACI), Novu handles all the communication plumbing — webhook ingestion, message normalization, conversation state, and delivery — so you can focus entirely on what your agent should do.

ACI draws a hard boundary between two concerns:

- **Communication infrastructure** — Webhook ingestion, message normalization, conversation state, identity resolution, and delivery across channels. ACI owns this layer.
- **Agent intelligence** — Your LLM, prompts, tools, managed agent configuration, business logic, or custom runtime. You own this layer.

The result: create an agent once and make it available across every connected channel. ACI routes messages in, sends replies out, preserves conversation context, and removes the need to rebuild communication plumbing for every platform.

## How a conversation flows

ACI turns channel-specific messages into a standard communication flow between the user and your agent.

<Steps>
  <Step title="User sends a message">
    A user sends a message on Slack, Microsoft Teams, WhatsApp, or another supported channel.
  </Step>
  <Step title="Novu ingests the webhook">
    Novu receives the platform webhook, normalizes the event, and resolves the user's identity against your subscriber records.
  </Step>
  <Step title="Novu forwards context to your agent">
    Novu creates or loads the conversation, assembles message history, and forwards the full context object to your agent brain.
  </Step>
  <Step title="Your agent processes and responds">
    Your agent processes the message — using an LLM, business logic, or managed platform — and produces a reply.
  </Step>
  <Step title="Novu delivers the reply">
    Novu posts the reply back to the correct platform thread, then persists the conversation and logs activity in the dashboard.
  </Step>
</Steps>

  <Card title="Managed Agent" icon="sparkles" href="/agents/managed-agent">

  </Card>
  <Card title="Custom Code Agent" icon="code" href="/agents/custom-code-agent">
    Write your own agent logic. Handle events, call any LLM or API, and reply using the Novu agent SDK. Compatible with AI SDK, LangChain, OpenAI SDK, and any custom code.

</CardGroup>
